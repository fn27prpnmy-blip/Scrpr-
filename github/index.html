📖 Kurzanleitung

1. PAT erstellen: github.com → Profil-Bild → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token.
Haken setzen bei: repo → Token kopieren.

2. Hier eintragen: Benutzername, Repo-Name und Token oben eingeben.

3. Website hinzufügen: Tab "Websites" → "+ Website hinzufügen".

4. Suche starten: Dashboard → "Suche jetzt starten".

Automatisch: Die GitHub Action läuft alle 6 Stunden selbstständig.
